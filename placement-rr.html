01 // Environment

Working in a live production environment

The challenge

The placement involved developing and supporting software systems inside a live automotive manufacturing environment where reliability, usability, traceability and operational impact mattered immediately.

Unlike university projects, solutions were deployed directly into production workflows and used daily by manufacturing teams and senior management across the plant.

The role combined software engineering, stakeholder management, systems thinking, manufacturing process understanding and rapid problem solving under operational constraints.

"Systems were expected to work reliably, integrate cleanly with existing processes, and provide measurable operational value."

02 // Major Project

Enterprise Asset Tracking System

Production Asset Tracking Platform

The largest project undertaken during the placement was the design and development of a comprehensive production asset tracking system built using Oracle APEX.

The system was developed collaboratively with another intern and involved the full software lifecycle:

  • Identifying operational problems across the production line.
  • Gathering requirements from stakeholders and manufacturing teams.
  • Designing database architecture and workflows.
  • Developing frontend and backend systems.
  • Deploying production-ready functionality to live users.
  • Providing documentation, training and support handover.

The platform tracks over 2,100 manufacturing jigs and tools across the assembly line using QR-code identification and live condition reporting.

A fully normalised Oracle database structure was designed and implemented, alongside a custom Zebra printer pipeline using RESTful print services, ZPL URL encoding and automated print queue handling.

The application is now used multiple times daily by over 100 concurrent users across the manufacturing environment.

Data from the platform feeds directly into live Power BI dashboards used for “Ready-to-Run” manufacturing decisions at multiple levels of management.

"The completed system generated a calculated operational saving of approximately £100,000 per year."

03 // Projects

Additional Work

Safety-Critical Data Analysis

Python Manufacturing Data

Following a safety-critical discovery relating to potentially loose fixings, I was tasked with analysing over 270,000 rows of historical production data covering more than 8 years of manufacturing records.

Due to internal restrictions, the analysis was implemented entirely in base Python without external libraries such as pandas. A custom CSV parsing and dictionary-based processing system was developed to execute complex validation logic efficiently.

The final solution completed analysis in approximately 7 seconds and was verified as accurate. The resulting insights contributed directly to vehicle recall investigations and safety review processes.

Operational Process Automation

Python Automation

Developed a custom automation system to replace a repetitive manual manufacturing workflow that previously consumed approximately two hours per day.

The automation interacted directly with Windows-level UI controls and used clipboard-based state interpretation and deterministic navigation logic to make operational decisions.

Robust error handling and Microsoft Teams webhook integration were implemented to automatically notify owners when unexpected conditions occurred.

Power Platform Solutions

Power Apps Power BI

Developed multiple production-facing Power Platform solutions, including operational startup systems, stock management tools and technical assessment applications.

Several applications extended SharePoint beyond native functionality using custom relational logic, bespoke interfaces and client-side workflows.

Supporting Power BI dashboards and reporting systems were also developed for manufacturing analytics, operational readiness and technical team analysis.
